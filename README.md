# Task Manager API

API REST para gestion de tareas usando node.js y express y guardada en la base de datos usando mySQL

## Funcionalidades

- Crear tareas
- Obtener tareas
- Editar tareas
- Eliminar tareas

## Tecnologias

 - Node.js
 - Express
 - Javascript
 - MySQL

## Instalacion
1. Clonar repositorio
2. Ejecutar 'npm install'
3. Crear archivo '.env' con las credenciales
4. Ejecutar 'node server.js'

## Endpoints
 - GET /tasks:id
 - POST /tasks:id
 - PUT /tasks/:id
